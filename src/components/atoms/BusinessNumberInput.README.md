# 사업자 번호 입력 컴포넌트 (Business Number Input Component)\n\n3개의 NumberInput을 사용하여 간단하고 직관적으로 구현된 사업자 번호 입력 컴포넌트입니다.\n\n## 🎯 설계 철학\n\n**\"간단함이 최고의 복잡함이다\"** - 불필요한 포맷팅 로직 없이 3개의 필드로 나누어 직관적인 사용자 경험을 제공합니다.\n\n## 📦 구조\n\n```\n📦 Simple BusinessNumber Architecture\n├── 🧩 Components\n│   ├── NumberInput.jsx              # 기본 숫자 입력 (변경 없음)\n│   └── BusinessNumberInput.jsx      # 3개 NumberInput 조합\n├── 🎨 Styles\n│   └── BusinessNumberInput.css      # 간단한 스타일\n└── 📖 StyleGuide\n    └── /dev/styleguide 페이지        # 사용 예시\n```\n\n## ✅ 장점\n\n### 1. **간단한 구현**\n- 복잡한 포맷팅 로직 불필요\n- 정규식이나 문자열 조작 없음\n- 각 필드별로 min/max 값으로 자동 제한\n\n### 2. **직관적인 사용자 경험**\n- 사용자가 자연스럽게 3-2-5 자리로 인식\n- 각 필드별로 명확한 입력 길이 제한\n- Tab 키로 자연스러운 필드 이동\n\n### 3. **유지보수 용이성**\n- 각 필드가 독립적으로 관리됨\n- 버그 발생 시 해당 필드만 수정\n- 새로운 기능 추가 시 영향도 최소화\n\n### 4. **성능 최적화**\n- 실시간 포맷팅으로 인한 리렌더링 없음\n- 각 필드별 개별 검증 가능\n- 메모리 사용량 최소화\n\n## 🚀 사용 방법\n\n### 기본 사용법\n\n```jsx\nimport React, { useState } from 'react';\nimport BusinessNumberInput from './components/atoms/BusinessNumberInput.jsx';\n\nfunction App() {\n  const [businessNumber, setBusinessNumber] = useState({\n    part1: '',\n    part2: '',\n    part3: ''\n  });\n  \n  const handleValidationComplete = (result) => {\n    console.log('검증 완료:', result);\n    if (result.isValid) {\n      alert(`유효한 사업자번호입니다!`);\n    }\n  };\n\n  return (\n    <BusinessNumberInput\n      label=\"사업자등록번호\"\n      value={businessNumber}\n      onChange={setBusinessNumber}\n      required\n      onValidationComplete={handleValidationComplete}\n    />\n  );\n}\n```\n\n### 자동 검증\n\n```jsx\n<BusinessNumberInput\n  label=\"사업자등록번호 (자동검증)\"\n  value={businessNumber}\n  onChange={setBusinessNumber}\n  autoValidate={true}          // 10자리 입력 시 자동 검증\n  showValidationButton={false} // 검증 버튼 숨김\n  onValidationComplete={handleValidationComplete}\n/>\n```\n\n## 🎛️ Props API\n\n| Prop | Type | Default | Description |\n|------|------|---------|-------------|\n| `value` | `{part1, part2, part3}` | `{part1:'', part2:'', part3:''}` | 각 필드의 값 |\n| `onChange` | `(value) => void` | - | 값 변경 콜백 |\n| `label` | `string` | `'사업자등록번호'` | 라벨 텍스트 |\n| `required` | `boolean` | `false` | 필수 입력 여부 |\n| `disabled` | `boolean` | `false` | 비활성화 여부 |\n| `error` | `boolean` | `false` | 에러 상태 |\n| `errorMessage` | `string` | - | 에러 메시지 |\n| `showValidationButton` | `boolean` | `true` | 검증 버튼 표시 여부 |\n| `showCompanyName` | `boolean` | `true` | 회사명 표시 여부 |\n| `autoValidate` | `boolean` | `false` | 자동 검증 여부 |\n| `onValidationComplete` | `(result) => void` | - | 검증 완료 콜백 |\n| `onValidationError` | `(error) => void` | - | 검증 에러 콜백 |\n\n## 🎨 스타일링\n\n### CSS 클래스\n\n```css\n.business-number-input-wrapper { }    /*
